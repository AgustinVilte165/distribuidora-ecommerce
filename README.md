Proyecto desarrollado por **Agustin Vilte**

---

##  Descripción

Ecommerce funcional con panel de administración desarrollado con Node.js y MongoDB Atlas.

El sistema permite:

- Registro y login de usuarios
- Autenticación con sesiones
- Creación de pedidos
- Historial de compras por usuario
- Panel de administrador para visualizar pedidos
- Conexión a base de datos en la nube (MongoDB Atlas)

---

##  Tecnologías utilizadas

- Node.js
- Express
- MongoDB Atlas
- Mongoose
- Express-session
- Bcrypt (encriptación de contraseñas)
- HTML / CSS / JavaScript

##  Instalación local

Clonar el repositorio:

```bash
git clone https://github.com/AgustinVilte165/distribuidora-ecommerce.git
```

Instalar dependencias:

```bash
npm install
```

Crear archivo `.env` con:

```
PORT=3000
MONGO_URI=TU_URI_DE_MONGO
SESSION_SECRET=TU_SECRET
```

Iniciar servidor:

```bash
npm start
```
